# PicZoomer 隐私声明

生效日期：2026 年 6 月 23 日

PicZoomer 不收集、存储、处理或传输个人信息、浏览记录或网站内容。

## 数据处理

- 不使用分析或遥测服务。
- 不包含广告或用户跟踪功能。
- 不使用 Cookie、`localStorage`、IndexedDB 或 Chrome Storage 保存数据。
- 不向开发者或第三方服务器发送网络请求。
- 不加载或执行远程代码。

## 网站访问

扩展仅在 `https://bsky.app/*` 上运行。该访问范围用于识别 Bluesky 的图片灯箱，并在用户操作图片时提供缩放和平移功能。

扩展不会读取、修改或传输账号凭据、私信、帖子内容或其他个人数据。

## 权限

扩展不申请 Chrome 扩展 API 权限。Manifest 中的网站匹配范围仅用于加载本地内容脚本。

## 第三方服务

使用 bsky.app 时，仍适用 Bluesky 自身的隐私政策。PicZoomer 是独立开发的第三方扩展，与 Bluesky Social PBC 无隶属、赞助或官方合作关系。

## 变更

如果未来版本引入数据处理行为，本声明将在发布前更新，并在适用时请求用户同意。
